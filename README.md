# Cide

A modern e-commerce WordPress theme built for online stores.

Cide ships with a built-in hero slider, product showcase sections, five header menu styles, smooth gradient transitions, dark mode, and deep WooCommerce integration. It is lightweight, accessible, and translation-ready.

## Features

- Built-in hero slider with Customizer controls (image, badge, title, text, button per slide)
- Five header menu styles: Pill, Underline, Minimal, Boxed, Gradient bar
- Theme color options: primary, secondary, and accent colors in the Customizer
- WooCommerce front-page sections: shop by category, featured products, new arrivals
- Trust badges strip (free shipping, secure checkout, easy returns, support)
- Responsive product grid (4 / 3 / 2 columns)
- Slide-in off-canvas mobile menu with overlay
- Dark mode support (follows the visitor's system preference)
- Fluid typography and smooth gradient transitions
- Accessible: skip links, screen-reader text, focus styles, reduced-motion support
- Translation-ready (text domain: `cide`)

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- WooCommerce (optional, for the e-commerce sections)

## Installation

1. Download the theme as a ZIP.
2. In WordPress admin, go to **Appearance > Themes > Add New > Upload Theme**.
3. Upload the ZIP and click **Install Now**, then **Activate**.
4. Go to **Appearance > Customize** to set up the hero slider, menu style, and colors.
