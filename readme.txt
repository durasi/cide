== Description ==

Cide is a modern e-commerce WordPress theme built for online stores. It ships with a built-in hero slider, product showcase sections, five header menu styles, smooth gradient transitions, dark mode, and deep WooCommerce integration. Lightweight, accessible, and translation-ready, Cide is made for shops, brands, and product-driven businesses.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and choose the cide.zip file, then click Install Now.
3. Click Activate to use the theme on your site.
4. Go to Appearance > Customize to set the hero slider, menu style, and other options.

== Frequently Asked Questions ==

= How do I set up the hero slider? =

Go to Appearance > Customize > Hero Slider. You can set an image, badge, title, text, and button for up to three slides. Leave a slide title empty to hide it.

= How do I change the menu style? =

Go to Appearance > Customize > Header & Menu and choose one of the five styles: Pill, Underline, Minimal, Boxed, or Gradient bar.

= Does Cide support WooCommerce? =

Yes. When WooCommerce is active, the front page automatically shows category, featured product, and new arrival sections, and all shop pages are styled to match the theme.

= Is Cide translation ready? =

Yes. All theme strings use the "cide" text domain and the theme loads its own language files from the /languages folder.
